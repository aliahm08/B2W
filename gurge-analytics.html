    Dashboard Guide

    This dashboard provides an interactive way to analyze bus fleet performance and reliability.

    Chart Visualization Styles

    You can change the main chart's appearance using the toggle buttons:

    • Expanded View: Shows every individual bus as a single point. Click a bus to see its detailed Performance Profile.
    • Cluster View: Groups buses into performance-based clusters using a K-Means algorithm. This helps identify high-level patterns and personas (e.g., "Reliable Workhorses", "Problematic Units"). Click a cluster to see the buses within it.
    • Matrix View: Divides the chart into a grid, clustering buses based on their performance quadrant. This offers a structured overview of the fleet's distribution.

    Interacting with the Dashboard

    • Fleet Panel: Use the left panel to search, sort, and select fleets. The color of each card represents its performance score (MDBF vs. age), and the sparkline shows the recent MDBF trend.
    • Details on Demand: Click on any bus in the main scatter plot to open its Performance Profile, which includes a radar chart for multi-metric comparison and a log of recent failures.
